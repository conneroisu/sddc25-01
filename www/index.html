Project Overview

Our Machine Learning: Semantic Segmentation Optimization project focuses on optimizing semantic segmentation algorithms for eye tracking in assistive technology applications. We aim to improve the performance of these algorithms for individuals with mobility disabilities, particularly those with conditions like cerebral palsy.

By pipelining a U-Net neural network algorithm into four equal parts that can run concurrently while maintaining accuracy, we aim to increase processing speed from 160 ms per frame to 33.2ms per frame for 4 frames simultaneously. This will increase throughput and maintain our required 99.8% IoU accuracy while achieving a target processing speed of 60 frames per second.

Key innovations in our approach include:

  • Parallel processing of the U-Net semantic segmentation algorithm
  • Efficient memory allocation and thread synchronization
  • Optimization for the AMD Kria KV260 development board
  • Strategic DPU (Deep Processing Unit) resource sharing
